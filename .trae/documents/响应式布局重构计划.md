# 前端响应式布局重构计划

## 1. 概述

基于 SuperDesign 设计规范，采用 **Mobile-First** 设计方法，使用 `min-width` 媒体查询实现响应式布局，同时保证 PC 端的优秀体验。

### 设计断点（完整覆盖）

| 断点      | 宽度      | 设备类型    |
| ------- | ------- | ------- |
| 默认      | < 480px | 小屏手机    |
| 480px+  | 480px+  | 手机/大屏手机 |
| 768px+  | 768px+  | 平板      |
| 1024px+ | 1024px+ | 小桌面     |
| 1280px+ | 1280px+ | 桌面      |

### 设计原则

* **触控优先**：移动端最小触控区域 44x44px

* **渐进增强**：从移动端开始，逐步为更大屏幕添加样式

* **内容优先**：移动端优先显示核心内容，非核心内容可隐藏

* **使用 CSS 变量系统** (`--spacing`, `--radius`)

* **动画过渡** 150-200ms ease

* **Inter 字体家族**

## 4. App.tsx 布局重构

### 4.1 修改文件
- `App.tsx`

### 4.2 核心逻辑

```tsx
const AppContent: React.FC = () => {
  const [activeTab, setActiveTab] = useState('collection');
  const [isMobile, setIsMobile] = useState(false);
  const [drawerOpen, setDrawerOpen] = useState(false);

  // 检测屏幕尺寸
  useEffect(() => {
    const checkMobile = () => setIsMobile(window.innerWidth < 768);
    checkMobile();
    window.addEventListener('resize', checkMobile);
    return () => window.removeEventListener('resize', checkMobile);
  }, []);

  const menuItems = [
    { key: 'collection', icon: <DownloadOutlined />, label: t.menu.collection },
    { key: 'management', icon: <DatabaseOutlined />, label: t.menu.management },
    { key: 'chart', icon: <LineChartOutlined />, label: t.menu.chart },
  ];

  return (
    <Layout className="app-layout">
      <Header className="app-header">
        <div className="logo">{t.app.title}</div>
        <div className="header-actions">
          {/* 语言/主题切换按钮 */}
        </div>
      </Header>

      <Layout>
        {/* PC端侧边栏 */}
        {!isMobile && (
          <Sider width={200} className="app-sider" collapsible>
            <Menu mode="inline" selectedKeys={[activeTab]} items={menuItems} onClick={({ key }) => setActiveTab(key)} />
          </Sider>
        )}

        {/* 移动端抽屉 */}
        {isMobile && drawerOpen && (
          <>
            <div className="drawer-overlay open" onClick={() => setDrawerOpen(false)} />
            <div className="app-sider-drawer open">
              <Menu mode="inline" selectedKeys={[activeTab]} items={menuItems} onClick={({ key }) => { setActiveTab(key); setDrawerOpen(false); }} />
            </div>
          </>
        )}

        <Content className="app-content">
          {activeTab === 'collection' && <DataCollection />}
          {activeTab === 'management' && <DataManagement />}
          {activeTab === 'chart' && <KlineChart />}
        </Content>
      </Layout>

      {/* 移动端底部Tab导航 */}
      {isMobile && (
        <div className="mobile-tab-nav">
          {menuItems.map(item => (
            <div
              key={item.key}
              className={`mobile-tab-item ${activeTab === item.key ? 'active' : ''}`}
              onClick={() => setActiveTab(item.key)}
            >
              {item.icon}
              <span>{item.label}</span>
            </div>
          ))}
        </div>
      )}
    </Layout>
  );
};
```
